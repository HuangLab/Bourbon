Padding

View source

The padding mixin accepts up to four values, including null, and uses the directional-property mixin to map them to their respective directions.

{% highlight scss %} @include padding(20vh null 10px 3em); {% endhighlight %}

CSS Output

{% highlight scss %} padding-top: 20vh; padding-bottom: 10px; padding-left: 3em; {% endhighlight %}
