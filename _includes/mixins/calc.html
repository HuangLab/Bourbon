Calc

View source View spec

A mixin for vendor-prefixing the CSS3 calc function. It accepts a property and a value.

{% highlight scss %} @include calc(width, "100% - 80px"); {% endhighlight %} Note: You must use interpolation to pass in a variable: #{…}. {% highlight scss%} $width: 100%; @include calc(width, "#{$width} - 80px"); {% endhighlight %}
