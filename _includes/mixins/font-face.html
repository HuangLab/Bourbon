Font Face

Generates an @font-face declaration. Accepts arguments for weight, style, usage with the Rails Asset Pipeline and file formats.

{% highlight scss %} @include font-face("source-sans-pro", "/fonts/source-sans-pro/source-sans-pro-regular"); @include font-face("source-sans-pro", "/fonts/source-sans-pro/source-sans-pro-bold", bold); @include font-face("source-sans-pro", "/fonts/source-sans-pro/source-sans-pro-italic", normal, italic); {% endhighlight %}

The $file-formats argument allows you specify your fontâ€™s file formats (the default is eot woff2 woff ttf svg:

{% highlight scss %} @include font-face("source-sans-pro", "/fonts/source-sans-pro/source-sans-pro-regular", $file-formats: eot woff2 woff); {% endhighlight %}

You can also use the Rails Asset Pipeline (place the fonts in app/assets/fonts/:

{% highlight scss %} @include font-face("source-sans-pro", "source-sans-pro/source-sans-pro-regular", normal, $asset-pipeline: true, $file-formats: eot woff ttf); {% endhighlight %}
